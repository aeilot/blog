TL;DR

你好，陌生人！我是 滿五，很高興見到你。

簡短介紹我自己，就是內向 + 理想主義，MBTI 是 INFP，但 MBTI 畢竟僅供參考，具體的個體肯定情況複雜。喜歡發呆，開小差，思考一些天馬行空的東西，有的時候會寫下來。有時自卑，有時自負，有時樂觀，有時悲觀。當然，總的來說還是願意認識來自四海八方的各位。聯絡方式：Twitter 以及 Telegram。

從很早以前就開始對計算機感興趣，於是自學了 C 語言和 Linux。後來認識了一些同學，比如 MrWillCom，於是對前端開發有了一些瞭解。空閒時間喜歡做一些開源專案，或者在 GitHub 上面水，比如初中的時候開發了一款 Android 應用，還參與了一些專案文件的翻譯工作。我目前是高中生，學校的技術顧問，負責幹苦力維護一些機選系統之類。高中之後划水專注於 OI 的學習，也混出了一些獎項。還有我的課題，我開始對 AI 有了初步的瞭解，未來可能從事相關的工作。

自己覺得自己挺文藝（bushi），也有人這樣說（。沒事幹喜歡看一些文學書籍/技術文獻，自己也嘗試過詩歌/散文的創作。小說？小說沒有想到什麼好的劇情（。也喜歡攝影，出遠門一定會帶相機的。學過一些設計/剪輯軟體的使用，導演過初中畢業的 MV（），夢想之一是拍微電影。放鬆的時候喜歡聽音樂，聽得很雜，古典/J-Pop/國際流行/Rock/電子，什麼都聽。喜歡 ヨルシカ、YOASOBI、ずっと真夜中でいいのに。（）自己會彈吉他，但是技術就那樣。自認為有一些語言天賦，在學習日語/西班牙語。英語在準備託福。總的來說是不愛運動的，但是有時還是會叫人出去騎車（。

CHANGELOG

2018 年之前，不知道什麼時候開始，我使用的是 Google Blogspot。

2018 年盛夏，我開發了本站的 1.0 版本，是完全自己編寫的。但是由於事情多，且對於前端開發不熟悉，沒幾天就放棄了。

接著，可能是 2019 年，我使用了 Jekyll 框架，但是，我很快就發現了它的弊端。Jekyll 很多地方不符合習慣，而且是 Ruby 寫的，而我不常用它。

過了幾個月，我找到了 Hexo。我曾經用過一段時間，還改了幾個主題，貢獻了幾行程式碼。但是，它實在是有點慢，而且外掛分散，要下好多東西。

2020 年夏季，我開始使用 Hugo。暫時我沒有發現什麼問題，而且生成迅速。

2021 年春季，由於 Hugo 主題缺少，我貨比三家之後回到了 Hexo。Hexo最近更新了 5.0，變得快了很多。。。

2023 年夏季開始，部分部落格將在知乎/xLog同步。
