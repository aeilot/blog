TL;DR

Code

2014 年夏季，一个二年级小学生偶然间看到了父亲电脑屏幕上接连不断的代码，对这个陌生的界面，这些奇妙的字符深深吸引了。

2015 年初，TA 给一台旧电脑装上了 Linux 系统，似乎是 Ubuntu，然后便入了门。很艰难，但还是坚持下来了。最开始学的是 C 语言，把各种数据结构全部自己动手实现了一下，感觉很有趣。

2017 年，TA 加入了 GitHub，并把自己的第一个项目放了上去。当时用的还是 这个账号。

不知什么时候，偶然间读了《乔布斯传》，很有感触，发奋立志要 Change the world!

2018 年初，TA 攒了钱，买了一台 iMac 21.5，用到现在，便入门了 iOS 开发。后来又因为各种原因，顺便学了学 Android，上架了自己的项目—— Daily Notes (现在已经停止维护)。

2019 年，如今的 TA 去参加各项比赛了！NOIP、WWDC……尝试、挑战、收获、结交好友，充满希望。

Qt、Electron、Cocoa、Vue……Java、Swift、C/C++…… 无数框架、语言，无数项目、练习，无数次自学时找资料，有了经验……不知不觉，似乎 6 年已经过去了。你还记得当时的理想吗？你还差得很远呢，目前还只是个菜鸡呢，加油吧！

Others

  • 2018 年，不知道怎么，去学了学 PR、AE、C4D、PS、AI、DaVinci 之类的东西，想拍点什么，记录下点什么。我很疑惑，至今却一直没看见发布过什么，只是自娱自乐。
  • 2020 年，突然对于文学很感兴趣，开始不断尝试与突破。

CHANGELOG

2018 年之前，不知道什么时候开始，我使用的是 Google Blogspot。

2018 年盛夏，我弄了本站的 1.0 版本，是完全自己编写的。但是由于事情多，且对于前端开发不熟悉，没几天就放弃了。

接着，可能是 2019 年，我使用了 Jekyll 框架，但是，我很快就发现了它的弊端。Jekyll 很多地方不符合习惯，而且是 Ruby 写的，而我不常用它。

过了几个月，我找到了 Hexo。我曾经用过一段时间，还改了几个主题，贡献了几行代码。但是，它实在是有点慢，而且插件分散，要下好多东西。

2020 年夏季，我开始使用 Hugo。暂时我没有发现什么问题，而且生成迅速。

2021 年春季，由于 Hugo 主题缺少，我货比三家之后回到了 Hexo。Hexo最近更新了 5.0，变得快了很多。。。
