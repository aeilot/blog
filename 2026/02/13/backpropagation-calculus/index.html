Backpropagation: A Vector Calculus Perspective

The primary goal of backpropagation is to calculate the partial derivatives of the cost function CCC with respect to every weight W\mathbf{W}W and bias b\mathbf{b}b in the network.

1. Notation (Matrix Form)

To facilitate a clean derivation using matrix calculus, we adopt the following conventions:

  • al\mathbf{a}^lal: Activation vector of the lll-th layer (nl×1n_l \times 1nl​×1).
  • zl\mathbf{z}^lzl: Weighted input vector of the lll-th layer (nl×1n_l \times 1nl​×1), where zl=Wlal−1+bl\mathbf{z}^l = \mathbf{W}^l \mathbf{a}^{l-1} + \mathbf{b}^lzl=Wlal−1+bl.
  • Wl\mathbf{W}^lWl: Weight matrix of the lll-th layer (nl×nl−1n_l \times n_{l-1}nl​×nl−1​).
  • bl\mathbf{b}^lbl: Bias vector of the lll-th layer (nl×1n_l \times 1nl​×1).
  • δl\boldsymbol{\delta}^lδl: Error vector of the lll-th layer, defined as δl≡∂C∂zl\boldsymbol{\delta}^l \equiv \frac{\partial C}{\partial \mathbf{z}^l}δl≡∂zl∂C​.
  • ⊙\odot⊙: Hadamard product (element-wise multiplication).
  • σ′(zl)\sigma'(\mathbf{z}^l)σ′(zl): The derivative of the activation function applied element-wise to zl\mathbf{z}^lzl.

2. Derivation of the Equations

BP1: Error at the Output Layer

Goal: δL=∇aLC⊙σ′(zL)\boldsymbol{\delta}^L = \nabla_{\mathbf{a}^L} C \odot \sigma'(\mathbf{z}^L)δL=∇aL​C⊙σ′(zL)

Derivation Steps:

  1. Apply the Chain Rule: The error δL\boldsymbol{\delta}^LδL represents how the cost CCC changes with respect to the weighted input zL\mathbf{z}^LzL. Since CCC depends on zL\mathbf{z}^LzL through the activations aL\mathbf{a}^LaL:

δL=∂C∂zL=(∂aL∂zL)T∂C∂aL\boldsymbol{\delta}^L = \frac{\partial C}{\partial \mathbf{z}^L} = \left( \frac{\partial \mathbf{a}^L}{\partial \mathbf{z}^L} \right)^T \frac{\partial C}{\partial \mathbf{a}^L} δL=∂zL∂C​=(∂zL∂aL​)T∂aL∂C​

  1. Compute the Jacobian: Because ajL=σ(zjL)a^L_j = \sigma(z^L_j)ajL​=σ(zjL​) (each output depends only on its own input), the Jacobian matrix ∂aL∂zL\frac{\partial \mathbf{a}^L}{\partial \mathbf{z}^L}∂zL∂aL​ is a diagonal matrix:

∂aL∂zL=diag(σ′(z1L),…,σ′(znL))\frac{\partial \mathbf{a}^L}{\partial \mathbf{z}^L} = \text{diag}(\sigma'(z^L_1), \dots, \sigma'(z^L_n)) ∂zL∂aL​=diag(σ′(z1L​),…,σ′(znL​))

  1. Result: Multiplying a diagonal matrix by a vector is equivalent to the Hadamard product:

δL=∇aLC⊙σ′(zL)\boldsymbol{\delta}^L = \nabla_{\mathbf{a}^L} C \odot \sigma'(\mathbf{z}^L) δL=∇aL​C⊙σ′(zL)

BP2: Propagating Error to Hidden Layers

Goal: δl=((Wl+1)Tδl+1)⊙σ′(zl)\boldsymbol{\delta}^l = ((\mathbf{W}^{l+1})^T \boldsymbol{\delta}^{l+1}) \odot \sigma'(\mathbf{z}^l)δl=((Wl+1)Tδl+1)⊙σ′(zl)

Derivation Steps:

  1. Link Consecutive Layers: We express the error at layer lll in terms of the error at layer l+1l+1l+1 using the multivariate chain rule:

δl=∂C∂zl=(∂zl+1∂zl)T∂C∂zl+1=(∂zl+1∂zl)Tδl+1\boldsymbol{\delta}^l = \frac{\partial C}{\partial \mathbf{z}^l} = \left( \frac{\partial \mathbf{z}^{l+1}}{\partial \mathbf{z}^l} \right)^T \frac{\partial C}{\partial \mathbf{z}^{l+1}} = \left( \frac{\partial \mathbf{z}^{l+1}}{\partial \mathbf{z}^l} \right)^T \boldsymbol{\delta}^{l+1} δl=∂zl∂C​=(∂zl∂zl+1​)T∂zl+1∂C​=(∂zl∂zl+1​)Tδl+1

  1. Differentiate the Linear Transformation: Recall zl+1=Wl+1al+bl+1\mathbf{z}^{l+1} = \mathbf{W}^{l+1} \mathbf{a}^l + \mathbf{b}^{l+1}zl+1=Wl+1al+bl+1 and al=σ(zl)\mathbf{a}^l = \sigma(\mathbf{z}^l)al=σ(zl). Applying the chain rule to find ∂zl+1∂zl\frac{\partial \mathbf{z}^{l+1}}{\partial \mathbf{z}^l}∂zl∂zl+1​:

∂zl+1∂zl=∂zl+1∂al⋅∂al∂zl=Wl+1⋅diag(σ′(zl))\frac{\partial \mathbf{z}^{l+1}}{\partial \mathbf{z}^l} = \frac{\partial \mathbf{z}^{l+1}}{\partial \mathbf{a}^l} \cdot \frac{\partial \mathbf{a}^l}{\partial \mathbf{z}^l} = \mathbf{W}^{l+1} \cdot \text{diag}(\sigma'(\mathbf{z}^l)) ∂zl∂zl+1​=∂al∂zl+1​⋅∂zl∂al​=Wl+1⋅diag(σ′(zl))

  1. Transpose and Simplify: Substitute back and use the property (AB)T=BTAT(AB)^T = B^T A^T(AB)T=BTAT:

δl=(Wl+1⋅diag(σ′(zl)))Tδl+1=diag(σ′(zl))(Wl+1)Tδl+1\boldsymbol{\delta}^l = \left( \mathbf{W}^{l+1} \cdot \text{diag}(\sigma'(\mathbf{z}^l)) \right)^T \boldsymbol{\delta}^{l+1} = \text{diag}(\sigma'(\mathbf{z}^l)) (\mathbf{W}^{l+1})^T \boldsymbol{\delta}^{l+1} δl=(Wl+1⋅diag(σ′(zl)))Tδl+1=diag(σ′(zl))(Wl+1)Tδl+1

  1. Final Form:
    Convert the diagonal matrix multiplication to a Hadamard product:

δl=((Wl+1)Tδl+1)⊙σ′(zl)\boldsymbol{\delta}^l = ((\mathbf{W}^{l+1})^T \boldsymbol{\delta}^{l+1}) \odot \sigma'(\mathbf{z}^l) δl=((Wl+1)Tδl+1)⊙σ′(zl)

BP3: Gradient for Biases

Goal: ∂C∂bl=δl\frac{\partial C}{\partial \mathbf{b}^l} = \boldsymbol{\delta}^l∂bl∂C​=δl

Derivation Steps:

  1. Chain Rule via Intermediate zzz:

∂C∂bl=(∂zl∂bl)T∂C∂zl\frac{\partial C}{\partial \mathbf{b}^l} = \left( \frac{\partial \mathbf{z}^l}{\partial \mathbf{b}^l} \right)^T \frac{\partial C}{\partial \mathbf{z}^l} ∂bl∂C​=(∂bl∂zl​)T∂zl∂C​

  1. Compute Local Gradient: From zl=Wlal−1+bl\mathbf{z}^l = \mathbf{W}^l \mathbf{a}^{l-1} + \mathbf{b}^lzl=Wlal−1+bl, we see that bl\mathbf{b}^lbl is added directly to the weighted sum. Thus, ∂zl∂bl\frac{\partial \mathbf{z}^l}{\partial \mathbf{b}^l}∂bl∂zl​ is the Identity matrix I\mathbf{I}I.

  2. Conclusion:

∂C∂bl=ITδl=δl\frac{\partial C}{\partial \mathbf{b}^l} = \mathbf{I}^T \boldsymbol{\delta}^l = \boldsymbol{\delta}^l ∂bl∂C​=ITδl=δl

BP4: Gradient for Weights

Goal: ∂C∂Wl=δl(al−1)T\frac{\partial C}{\partial \mathbf{W}^l} = \boldsymbol{\delta}^l (\mathbf{a}^{l-1})^T∂Wl∂C​=δl(al−1)T

Derivation Steps:

  1. Element-wise Approach: Consider a single weight wjklw^l_{jk}wjkl​ connecting neuron kkk in layer l−1l-1l−1 to neuron jjj in layer lll:

∂C∂wjkl=∂C∂zjl∂zjl∂wjkl=δjl∂zjl∂wjkl\frac{\partial C}{\partial w^l_{jk}} = \frac{\partial C}{\partial z^l_j} \frac{\partial z^l_j}{\partial w^l_{jk}} = \delta^l_j \frac{\partial z^l_j}{\partial w^l_{jk}} ∂wjkl​∂C​=∂zjl​∂C​∂wjkl​∂zjl​​=δjl​∂wjkl​∂zjl​​

  1. Solve for Partial Derivative: Since zjl=∑mwjmlaml−1+bjlz^l_j = \sum_m w^l_{jm} a^{l-1}_m + b^l_jzjl​=∑m​wjml​aml−1​+bjl​, the derivative with respect to wjklw^l_{jk}wjkl​ is simply akl−1a^{l-1}_kakl−1​. Therefore, ∂C∂wjkl=δjlakl−1\frac{\partial C}{\partial w^l_{jk}} = \delta^l_j a^{l-1}_k∂wjkl​∂C​=δjl​akl−1​.

  2. Vectorize as an Outer Product: The collection of all such derivatives δjlakl−1\delta^l_j a^{l-1}_kδjl​akl−1​ for all j,kj, kj,k forms the Outer Product of the error vector δl\boldsymbol{\delta}^lδl and the input activation vector al−1\mathbf{a}^{l-1}al−1:

∂C∂Wl=δl(al−1)T\frac{\partial C}{\partial \mathbf{W}^l} = \boldsymbol{\delta}^l (\mathbf{a}^{l-1})^T ∂Wl∂C​=δl(al−1)T
